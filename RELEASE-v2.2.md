## ⚠️ 已知问题

### 1️⃣ cron API 集成限制

**问题**: `brain_auto_remind.py` 检测后无法自动创建 cron 任务

**原因**: 
- `npx openclaw cron add` 命令行参数解析有问题
- 需要额外的 `--name` 参数
- JSON 参数转义复杂

**临时方案**:
- 自动检测功能正常
- cron 任务标记为"pending"
- 用户可以手动运行 cron 命令创建

**长期方案**:
- 使用 OpenClaw cron 工具 API（不是命令行）
- 或配置 OpenClaw 钩子自动创建

**影响**: 低（检测功能正常，只是 cron 创建需要手动）

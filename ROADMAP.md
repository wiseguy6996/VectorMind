## 📝 开发笔记

### 遗忘曲线算法
```python
# 艾宾浩斯遗忘曲线
# 复习间隔：1 天、3 天、7 天、15 天、30 天

def get_review_time(memory):
    recall_count = memory.recall_count
    last_recalled = memory.last_recalled
    
    if recall_count == 0:
        return last_recalled + 1  # 1 天后首次复习
    elif recall_count == 1:
        return last_recalled + 3  # 3 天后
    elif recall_count == 2:
        return last_recalled + 7  # 7 天后
    elif recall_count == 3:
        return last_recalled + 15  # 15 天后
    else:
        return last_recalled + 30  # 30 天后
```

### 记忆蒸馏算法

# 按重要度和检索次数筛选核心记忆
def distill_memories(days=7):
    memories = get_memories_by_date(days)
    
    # 筛选条件
    core = [m for m in memories if m.importance >= 8]

    # 生成摘要

        'core': core,

    }

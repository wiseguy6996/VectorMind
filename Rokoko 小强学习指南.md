## 📦 第一步：下载 Rokoko Studio Live

### 1. 下载 Rokoko Studio
```
官网：https://www.rokoko.com/
下载：Rokoko Studio（免费版本可用）
```

### 2. 下载 Blender 插件
```
GitHub: https://github.com/Rokoko/rokoko-studio-live-blender
或：Rokoko 官网下载插件
```

### 3. 安装到 Blender
```
1. 打开 Blender
2. Edit → Preferences → Addons
3. Install... → 选择下载的 ZIP 文件
4. 启用 "Rokoko Studio Live" 插件
```

## 🎬 第二步：学习 Rokoko 骨骼绑定

### 核心学习点
| 学习内容 | 说明 | 解决小强的问题 |
|---------|------|-------------|
| **骨骼层级结构** | root → hips → spine → ... | 骨盆脱离问题 |
| **IK/FK 切换** | 一键切换 IK/FK 驱动 | 手脚僵硬问题 |
| **Pole Target** | 自动计算手肘/膝盖方向 | 弯曲方向问题 |
| **自动重定向** | 不同骨骼系统映射 | 数据不匹配问题 |
| **肌肉系统** | 自动肌肉隆起 | 手臂弧度问题 |

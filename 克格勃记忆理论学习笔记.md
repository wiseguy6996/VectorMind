## 📖 核心理论摘要

### 1. 记忆类型（现代心理学）

| 类型 | 持续时间 | 说明 |
|------|---------|------|
| **感觉记忆** | <0.5 秒 | 感官直接获取的信息 |
| **短期记忆** | 几分钟到几小时 | 需要关注的信息 |
| **长期记忆** | 多年 | 重要的信息，可永久存储 |

**对记忆大王的启示**：
- v1.0 已实现长期记忆（ChromaDB 向量存储）
- v2.0 需要加强短期记忆→长期记忆的转换（智能复习）

---

### 2. 遗忘曲线（艾宾浩斯）

**核心发现**：
- 遗忘在学习之后立即开始
- 最初遗忘速度很快，后来逐渐缓慢
- 复习可以显著减缓遗忘

**复习间隔建议**：
```
首次学习 → 1 天后复习 → 3 天后 → 7 天后 → 15 天后 → 30 天后
```

**对记忆大王 v2.0 的应用**：
```python
# 遗忘曲线算法
def get_review_time(memory):
    recall_count = memory.recall_count
    
    if recall_count == 0:
        return 1   # 1 天后首次复习
    elif recall_count == 1:
        return 3   # 3 天后
    elif recall_count == 2:
        return 7   # 7 天后
    elif recall_count == 3:
        return 15  # 15 天后
    else:
        return 30  # 30 天后
```

### 3. 图像记忆（核心方法）

**核心原理**：
> "人类的大脑非常善于记忆图像信息。因此，大多数记忆信息的技巧，即助记方法，都以我们的想象力为基础，将抽象的言语和数字信息转化为图像。"

**对记忆大王 v2.0 的应用**：
- 记忆可视化（关系图谱）
- 标签图像化（不同标签用不同颜色/图标）
- 重要度视觉提示（高重要度记忆突出显示）

---

### 4. 联想记忆（核心方法）

**核心原理**：
> "一个成功的间谍必须有能力留心他看到和听到的重要细节，重新诠释这些信息，并将它们与之前掌握的信息联系起来。"

**联想记忆两大原则**：
1. **联系** - 新信息与已有信息建立关联
2. **图像** - 抽象概念转化为具体图像

**对记忆大王 v2.0 的应用**：
- 记忆关系图谱（语义关联 + 标签共享）
- 智能推荐（"检索此记忆的人也看了..."）
- 自动标签（AI 分析记忆内容，自动添加标签）

---

### 5. 地点法（记忆宫殿）

**核心方法**：
- 把信息放在熟悉的地点
- 通过回忆地点来回忆信息
- 适合记忆大量有序信息

**对记忆大王 v2.0 的应用**：
- 记忆分类存储（6 大类：三人团、项目、技术、决策、教训、文档）
- 项目进度可视化（"记忆宫殿"式布局）
- 时间线视图（按日期浏览记忆）

---

### 6. 泽伊加尔尼克效应（Zeigarnik Effect）

**核心发现**：
> "未完成的任务比已完成的任务更容易被记住"

**对记忆大王 v2.0 的应用**：
- 待办事项优先推送
- 未完成项目置顶显示
- 进度提醒（"螳螂拳骨骼绑定进行中"）

---

### 7. 序列位置效应（Serial Position Effect）

**核心发现**：
- 人们更容易记住序列的开头和结尾
- 中间部分容易遗忘

**对记忆大王 v2.0 的应用**：
- 重要记忆置顶
- 最近记忆突出显示
- 中间记忆增加复习频率

---

### 8. 专注力训练

**核心方法**：
- 有选择地感知信息
- 忽略干扰项
- 长时间专注于指定事物

**对记忆大王 v2.0 的应用**：
- 小强专属推送（只推送相关记忆）
- 项目视图过滤（只看当前项目相关）
- 搜索优化（快速定位目标记忆）

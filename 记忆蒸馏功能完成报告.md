## 🎯 核心功能

### 1️⃣ 自动摘要生成
- 按日期过滤记忆
- 自动分类（按标签）
- 提取关键词
- 生成 Markdown 报告

### 2️⃣ 核心记忆提炼
- 筛选重要度≥9 的记忆
- 按重要度排序
- 生成核心记忆清单
- 永久保存

### 3️⃣ 统计分析
- 总记忆数
- 核心记忆数
- 平均重要度
- 每日记忆分布（可视化条）

### 4️⃣ 分类展示
- 按标签自动聚类
- 每类展示前 5 条
- 支持自定义分类数量

## 💡 使用场景

### 场景 1：每日回顾
```bash
# 每晚睡前生成今日摘要
python brain_distill.py daily

# 回顾今天做了什么
```

### 场景 2：周报生成
```bash
# 每周日生成周报
python brain_distill.py weekly

# 回顾一周进展
```

### 场景 3：月度总结
```bash
# 每月最后一天生成月报
python brain_distill.py monthly

# 回顾一月成果
```

### 场景 4：核心记忆检索
```bash
# 提炼核心记忆
python brain_distill.py core

# 快速找到最重要的记忆
```
